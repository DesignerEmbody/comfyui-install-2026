# Быстрый старт — Comfyui Install 2026

1. **Двойной клик** по **`comfyui-install-2026.exe`** в корне репозитория  
   или запустите **`START.bat`**
2. Либо **[Releases](../../releases/latest)** → скачайте тот же **`comfyui-install-2026.exe`**
3. Windows может спросить SmartScreen — «Подробнее» → «Выполнить»

> Файл **`comfyui-install-2026.exe`** — установщик/лаунчер под это репозиторий (comfyui-install-2026).
